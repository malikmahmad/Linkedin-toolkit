# 💼 LinkedIn Content Toolkit

A free, single-file web app with **300 ready-to-use LinkedIn post ideas** — complete with sample captions, content types, effort levels, and engagement tips. No installation, no backend, no dependencies.

---

## ✨ Features

- **300 post ideas** spanning 11 content categories
- **Sample captions** with a one-click copy button for each idea
- **Filter by** category, content type, effort level, and posting frequency
- **Full-text search** across titles, descriptions, and captions
- **Responsive design** — table view on desktop, card view on mobile
- **Dark theme** with a clean, minimal UI
- **100% offline** — one HTML file, zero dependencies

## 🚀 Getting Started

No setup required. Just open the file in any browser:

```
index.html → double-click or drag into your browser
```

That's it.

---

## 🔍 How to Use

1. **Search** — type any keyword in the search bar to filter across all columns
2. **Filter** — narrow results by category, content type (Text, Video, Infographic, Poll), effort (High / Medium / Low), or frequency
3. **Browse** — scroll through the table (or cards on mobile) to find ideas that fit your brand
4. **Copy** — hit the **Copy** button on any row to grab the sample caption straight to your clipboard
5. **Reset** — click **✕ Reset** to clear all filters at once

## 📋 Columns Explained

| Column | Description |
|---|---|
| # | Idea number (1–300) |
| Category | Top-level content category |
| Sub-category | More specific topic area |
| Short Title | Name of the post idea |
| Description | What the post is about |
| Why Publish? | The strategic value of posting it |
| Content Type | Text Post, Video, Infographic, or Poll |
| Effort | How much work it takes to produce |
| Target Audience | Who will get the most value from this |
| Frequency | How often you should post this type |
| How to Engage | Tips to spark comments and interaction |
| Visual | Suggested visual or graphic style |
| Sample Caption | A ready-to-use caption with hashtags |

## 📱 Mobile Support

On screens under 768px, the table automatically switches to a **card layout** — each idea displayed as a standalone card with all key details and the copy button.

---

## 🛠 Customization

Since it's a single HTML file, everything is self-contained and easy to modify:

- **Add ideas** — extend the `DATA` array in the `<script>` tag
- **Change colors** — edit the CSS variables in `:root` at the top of `<style>`
- **Add columns** — update the `<thead>` and the row-rendering logic in the JS
- **Rebrand** — swap out the logo icon, title, and tagline in the `.header` section
